# Restaurant Management System Frontend

This is the frontend part of the Restaurant Management System project built using React. The application allows users to manage menu items, place orders, book tables, and generate bills.

## Features

- **Menu Management**: Create, read, update, and delete menu items.
- **Order Management**: Place new orders and view existing orders.
- **Table Booking**: Reserve tables for customers.
- **Bill Generation**: Generate bills with GST calculations.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the frontend directory:
   ```
   cd frontend
   ```

3. Install the dependencies:
   ```
   npm install
   ```

## Usage

To start the frontend application, run:
```
npm start
```
This will launch the application in your default web browser.

## API Integration

The frontend communicates with the backend API for data management. Ensure that the backend server is running to access the full functionality of the application.
